This folder contains external models used in the simulator.

3D tree model :
- has been downloaded from https://free3d.com/3d-model/trees-9-53338.html
- has been converted to collada file format (.dae) using https://3d-convert.com/en/convert/3ds-to-dae.html/

Textures have been downloaded from https://www.all3dfree.net/

At the time of downloading, these files were licensed as "not allowed for commercial use".
